# TrendSnap
![Latest release](https://img.shields.io/github/v/release/owner/trendsnap)

Micro-SaaS que recomenda melhores horários e hashtags para Threads.

## Requisitos
- Node 18 LTS
- Yarn 4

## Instalação
```bash
yarn install
```

## Desenvolvimento
```bash
yarn dev
```

## Testes
```bash
yarn test
```

## Docker
```bash
docker-compose up --build
```
